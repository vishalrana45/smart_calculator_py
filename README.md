# 🧮 Smart Calculator

A simple command-line calculator built in Python that combines multiple everyday calculators into one tool.

Instead of juggling different apps for your daily math — checking a discount, working out your BMI, or converting units — Smart Calculator brings 7 handy tools together in a single, lightweight menu-driven program. Just run it, pick a number, and get your answer.

## ✨ Features

- 🔢 **Basic Calculator** — addition, subtraction, multiplication, division
- 🎂 **Age Calculator** — calculate age from date of birth
- 📊 **Percentage Calculator** — quick percentage calculations
- ⚖️ **BMI Calculator** — calculate Body Mass Index from height and weight
- 📏 **Unit Converter** — convert between common units
- 🏷️ **Discount Calculator** — calculate final price after discount
- 💡 **Electricity Bill Calculator** — estimate electricity bill from usage

## 🛠️ Technologies Used

- Python 3

## ▶️ How to Run

1. Clone this repository:
   
   ```bash
   git clone https://github.com/vishalrana45/smart_calculator_py.git
   ```
1. Navigate into the project folder:
   
   ```bash
   cd smart_calculator_py
   ```
1. Run the program:
   
   ```bash
   python smart_cal.py
   ```

## 📖 Usage

After running the program, choose an option from the menu (1–8) to access the calculator you need. Select option 8 to exit.
